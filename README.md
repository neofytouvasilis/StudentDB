StudentHubDB is a beginner-friendly MySQL database project designed to manage students, courses, and enrollments.
It demonstrates relational database design using primary keys, foreign keys, and JOIN operations.
The project includes sample data and useful SQL queries for practice.
It is ideal for beginners learning MySQL and database fundamentals.

## Database Structure
The project consists of three main tables:
students – stores student information
courses – stores course details
enrollments – links students with courses

## Project Files
schema.sql → Creates all database tables
data.sql → Inserts sample data
queries.sql → Useful SQL queries for practice
